# diff-utils

[![npm version](https://img.shields.io/npm/v/diff-utils.svg)](https://www.npmjs.com/package/diff-utils)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Simple diff utilities — find added/removed/changed items

## Installation

```bash
npm install diff-utils
```

## Functions

- `diffArrays` — Find added and removed items between two arrays
- `diffObjects` — Find changed keys between two objects
